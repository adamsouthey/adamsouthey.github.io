Project 4

ReactJS

The Mern Stack App

For my final project I have built an application using ReactJS. It aims to connects charities with local food retailers so surplus food can be shared with those less fortunate. I planned for my project using wireframes and Trello. I incorporated Google Maps and Google Autocomplete APIs to display the location of both food retailers and local charities. Users registering as a charity are able to see their location on a map and view nearby food retailers along with their contact information within a 3000m radius . I styled my application using Bootstrap and tested front end using mocha and chai.

Project 3

Angular

The Mean Stack App

The third project aimed to create an online platform for users to find meetup groups at their chosen location to meet people with similar interests. The application supports worldwide searches and displays their chosen meetup location on a map. We built this application using AngularJS. This project required planning using wireframes and Trello and a high degree of communication as we were committing to the same Git repository. I was particularly proud of the the ability for users to cache their meetup search entries. I was also happy with the styling for this project as we kept the design consistent across the website and made it fully responsive. For styling, we used the Bulma framework.

Project 2

Node.js

Google Maps App with an API

For my second project I created a website to be used by dive enthusiasts where they can share their recent dive experiences, the location of their dive and provide a small review. For this project I used Express.js and MongoDB. I consumed the Google Maps API and gave users the ability to provide information about their dive. My main index page displayed logged dives using a grid system from Bootstrap. A user could then click on an individual dive to find out more information.

GitHub Heroku

Project 1

Javascript

Build a Javascript Game

For my first project I created a game using HTML5, CSS, JavaScript and jQuery. I built an interactive card matching memory game where users must find cards with matching numbers to score a point and remove that pair from the board. I included 5 levels of difficulty (decreased time with increased difficulty). I styled my application with CSS and implemented the functionality for a user to restart a current level if they failed to find the pairs.
